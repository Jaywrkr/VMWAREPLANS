🏗️ Resumen de los tres planes

Broadcom simplificó el portafolio de VMware en tres ofertas principales. Cada una cubre un perfil distinto de necesidades.

Nivel Base

VMware vSphere Foundation (VVF)

Plataforma de workloads enterprise para infraestructura moderna. Entrega virtualización, gestión simplificada y eficiencia de costos. Sirve además como la base técnica sobre la que se construye VCF.

  • VMs y contenedores en paralelo con Kubernetes nativo
  • Gestión inteligente de operaciones e infraestructura
  • Infraestructura hiperconvergida (compute + storage)
  • SKU único para despliegue y escalabilidad simplificados
  • vSphere + vSphere Kubernetes Service + VCF Operations + vSAN
Edge Optimizado

VMware Cloud Foundation Edge (VCFE)

Configuración optimizada de VMware Cloud Foundation diseñada específicamente para casos de uso en el borde (edge). Trae la pila completa de VCF en un footprint compacto para sitios remotos o locaciones de baja latencia.

  • Stack completo de VCF adaptado para edge
  • Incluye NSX para networking definido por software
  • Gestión de flota integrada con VCF Operations
  • Soporte para clústeres de 2 nodos
  • Ideal para retail, manufactura y oficinas remotas
Full Stack / Nivel Máximo

VMware Cloud Foundation (VCF)

Plataforma de nube privada que combina la agilidad del cloud público con la seguridad y rendimiento on-premises. IaaS completo con automatización integrada y menor TCO.

  • IaaS completo: cómputo, storage, red, Kubernetes, seguridad
  • Automatización self-service para VMs y contenedores
  • Escalamiento sin incremento de personal
  • Automatización Day 0, Day 1 y Day 2
  • SKU único para despliegue full stack
