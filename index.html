🏗️ Resumen de los tres planes

Broadcom simplificó el portafolio de VMware en tres ofertas principales.

Nivel Base

VMware vSphere Foundation (VVF)

Plataforma de workloads enterprise para infraestructura moderna. Virtualización, gestión simplificada y eficiencia de costos.

  • VMs y contenedores con Kubernetes nativo
  • Gestión inteligente de operaciones e infraestructura
  • Infraestructura hiperconvergida (compute + storage)
  • vSphere + VKS + VCF Operations + vSAN
Edge Optimizado

VMware Cloud Foundation Edge (VCFE)

Configuración optimizada de VCF para casos de uso en el borde (edge). Stack completo en footprint compacto para sitios remotos.

  • Stack completo de VCF para edge
  • NSX para networking definido por software
  • Soporte para clústeres de 2 nodos
  • Ideal para retail, manufactura y oficinas remotas
Full Stack / Nivel Máximo

VMware Cloud Foundation (VCF)

Plataforma de nube privada con IaaS completo, automatización integrada y menor TCO.

  • IaaS completo: cómputo, storage, red, Kubernetes, seguridad
  • Automatización self-service para VMs y contenedores
  • Automatización Day 0, Day 1 y Day 2
  • SKU único para despliegue full stack
